1. 在使用git或者gitee的过程中，免不了使用公钥，那么对于配置多个ssh公钥的问题可以参考

   https://gitee.com/help/articles/4229#article-header0

2. 并且使用如下命令，可以进行切换

```javascript
ssh -T git@gitee.com
ssh -T git@github.com
```
